# Restaurant Menu

Page de menu de restaurant congolais réalisée avec HTML5 sémantique et images locales pour présenter entrées, plats, desserts, boissons, tarifs et réservation.

## Objectif

Ce projet répond au livrable du menu de restaurant du module HTML de la Phase 2. L'objectif est de fournir un contenu structuré, accessible et complet sans CSS externe.

## Fonctionnalités

- Navigation multi-sections (Entrées, Plats, Desserts, Boissons, Prix, Réservation)
- Section spécial du jour
- Articles de menu avec nom, description, prix et ingrédients
- Tableau de prix par catégorie
- Horaires d'ouverture structurés avec liste de définitions
- Formulaire de réservation accessible
- Footer de contact avec email et téléphone

## Technologies

- HTML5 sémantique
- Balises ARIA et accessibilité
- Images locales

## Structure
